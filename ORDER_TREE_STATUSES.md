# Статусы дерева заказов в MainForm

Файл фиксирует состав узлов в левом дереве заказов (`Forms/MainForm.cs`).

1. `Все задания` — все задания во всех статусах.
2. `Обработанные` — файлы, которые поступили в работу.
3. `В архиве` — задания, перенесенные в архив.
4. `Обрабатывается` — файлы, которые копируются в программу при первом поступлении, и заказы, которые собираются на печатный спуск (основная работа).
5. `Завершено` — спуск готов.
