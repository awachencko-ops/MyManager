## 6. Краткий чеклист заполнения

1. Открыть `Repository -> Settings -> General`.
2. В блоке `About` вставить описание из раздела 1 (или 2/3).
3. В `Topics` добавить теги из раздела 4 по одному.
4. По необходимости заполнить `Website`.
5. Сохранить изменения.
